# Icônes de l'extension

Placez vos icônes PNG dans ce dossier :

- `icon-16.png` : 16x16 pixels
- `icon-32.png` : 32x32 pixels
- `icon-48.png` : 48x48 pixels
- `icon-128.png` : 128x128 pixels

## Génération des icônes

Vous pouvez utiliser un service en ligne comme :
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Ou créer vos icônes manuellement avec un logiciel de design (Figma, Photoshop, etc.).

## Suggestion de design

L'icône devrait représenter un assistant IA, par exemple :
- Un robot ou bot
- Une bulle de dialogue avec une étoile
- Un cerveau stylisé
- Les lettres "AI" dans un cercle

Couleurs suggérées : violet (#667eea), rose (#764ba2) pour correspondre au thème de l'application.
