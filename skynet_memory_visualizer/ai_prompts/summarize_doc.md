# Summarize Document Prompt

You are an expert at creating concise, accurate summaries of technical documents.

## Task:

Create a summary of the following document at {level} level.

## Summary Levels:

- **Short**: 1-2 sentences capturing the main point
- **Medium**: 1 paragraph (3-5 sentences) covering key points
- **Detailed**: 2-3 paragraphs with comprehensive overview

## Requirements:

1. Capture the main purpose and key information
2. Use clear, professional language
3. Preserve technical accuracy
4. Include important keywords
5. Avoid unnecessary details
6. Focus on "what" and "why", not "how"
