#### Core Features
- **Session Management**: Create and manage multiple isolated browser contexts

- **Element Interaction**: Click, type, hover, drag-and-drop, file uploads
- **Form Automation**: Intelligent form filling with auto-submit
- **Data Extraction**: Text, attributes, HTML, links, images extraction
- **Screenshots & PDFs**: High-quality page capture and PDF generation
- **Waiting Mechanisms**: Smart waits for elements, navigation, and custom conditions

#### Advanced Features
- **Device Emulation**: Test on iPhone, iPad, Pixel, and other devices
- **Geolocation**: Set custom GPS coordinates
- **User Agent**: Custom user agent strings
- **Resource Blocking**: Block images, CSS, scripts for faster browsing
- **Console Logs**: Capture browser console output
- **Network Logs**: Monitor all network requests
- **Request Interception**: Block or modify network requests
- **JavaScript Evaluation**: Execute custom JS in page context
- **Cookie Management**: Set, get, and clear cookies
- **Session Persistence**: Save and restore authenticated sessions
